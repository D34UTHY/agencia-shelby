# Agência Shelby
Uma landing page fictícia com base em uma já feita por mim à pedido de um cliiente.
Como prova de um serviço feito por mim, decidi utilizar a grande parte do código que utilizei para a landing page de um cliente, nessa página que ficará no github. Foram alterados somente nome da empresa e imagem de fundo. O restante da página segue funcionalidades e design semelhante às feitas por mim anteriormente. Esse modelo está mais simples, sem outras páginas ou rolagem vertical. Somente a página principal como modelo.
